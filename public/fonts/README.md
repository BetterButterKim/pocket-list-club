# 폰트 파일 넣는 곳

슬랙으로 올라온 사진을 서버에서 합성할 때 이 폰트들을 사용해요.
아래 3개 파일을 이 폴더에 넣어주세요. (전부 무료, 상업적 사용 가능)

1. **BlackHanSans-Regular.ttf** — 제목용 (두꺼운 한글)
   https://fonts.google.com/specimen/Black+Han+Sans → "Get font" → "Download all"

2. **AlfaSlabOne-Regular.ttf** — 로고/날짜용
   https://fonts.google.com/specimen/Alfa+Slab+One → "Get font" → "Download all"

3. **NotoSansKR-Bold.ttf** — 본문 보조용
   https://fonts.google.com/noto/specimen/Noto+Sans+KR → "Get font"

각 zip을 풀면 .ttf 파일이 있어요. 파일명을 위와 정확히 똑같이 맞춰서 넣으면 됩니다.
(파일이 없어도 앱은 돌아가지만, 슬랙 인증 사진의 글씨가 기본 폰트로 나와요.
 웹앱에서 올린 사진은 브라우저 폰트를 쓰므로 영향 없습니다.)
